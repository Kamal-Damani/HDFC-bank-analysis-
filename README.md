# 🏦 HDFC Banking Analytics Dashboard (Power BI)

## 📊 Project Overview

This project analyzes banking data modeled after HDFC Bank using Power BI. The dashboard provides insights into customer demographics, transaction trends, loan distribution, credit risk, and overall financial performance.

The goal of this project is to demonstrate data analysis, visualization, and business insight generation using Power BI.

---

## 🛠 Tools & Technologies Used
- Power BI  
- Power Query  
- DAX (Data Analysis Expressions)  
- Microsoft Excel  
- Data Visualization & Business Intelligence  

---

## 📂 Dataset Information

The dataset contains 10,000+ banking records with the following attributes:

- Customer ID  
- Customer Name  
- Gender  
- Age  
- City & State  
- Account Type  
- Account Balance  
- Transaction ID  
- Transaction Date  
- Transaction Type  
- Transaction Amount  
- Loan Type  
- Loan Amount  
- Credit Score  

---

## 📈 Dashboard Pages

### 1️⃣ Customer Demographics Overview  
Displays customer distribution and segmentation.

**Insights:**
- Customers by state and city  
- Gender distribution  
- Age group segmentation  
- Account type distribution  

---

### 2️⃣ Transaction Analysis  
Analyzes transaction patterns and trends over time.

**Insights:**
- Monthly transaction trends  
- Transaction type distribution  
- Top cities by transaction value  
- Daily transaction activity  

---

### 3️⃣ Loan & Credit Risk Analysis  
Evaluates loan distribution and customer creditworthiness.

**Insights:**
- Loan type distribution  
- Loan amount by state  
- Credit score segmentation  
- Loan distribution by age group  

---

### 4️⃣ Bank Performance Overview  
Provides a high-level view of financial performance.

**Insights:**
- Total customers, balance, loans, transactions  
- Deposits vs loans comparison  
- Top customers by account balance  
- Customer value segmentation  
- Transaction contribution by account type  

---

## 📊 Key Insights
- Majority of customers belong to the middle-age segment  
- Savings accounts are the most preferred  
- Metro cities generate the highest transaction volume  
- High credit score customers receive larger loans  
- A small group of customers contributes significantly to revenue  

---

## 🚀 How to Use
1. Download the `.pbix` file from the dashboard folder  
2. Open it in Power BI Desktop  
3. Explore the interactive dashboards using slicers and filters
